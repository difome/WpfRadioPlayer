# Radio Player

WPF приложение для воспроизведения интернет-радиостанций с использованием Bass.dll.

## Функции

- Воспроизведение радиостанций из JSON файла
- Получение метаданных (название трека и исполнитель) в реальном времени
- Вкладки "Станции" и "Любимые"
- Сохранение любимых станций в файл settings.json
- Темный дизайн интерфейса
- Размер окна: 700x500 пикселей

## Структура

- `stations.json` - список всех радиостанций
- `settings.json` - настройки (любимые станции)

## Зависимости

- .NET 8.0
- ManagedBass (обертка для Bass.dll)
- Newtonsoft.Json

## Использование

1. Запустите приложение
2. Выберите станцию из списка для воспроизведения
3. Используйте кнопку ★ для добавления/удаления из любимых
4. Переключайтесь между вкладками "Станции" и "Любимые"
