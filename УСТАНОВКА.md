# Установка .NET для Windows

## Вариант 1: Установить .NET Desktop Runtime (только для запуска)

1. Откройте браузер и перейдите по ссылке:
   https://dotnet.microsoft.com/download/dotnet/8.0

2. Найдите раздел **".NET Desktop Runtime 8.0.x"**

3. Скачайте версию для Windows x64:
   - Файл будет называться примерно: `windowsdesktop-runtime-8.0.x-win-x64.exe`

4. Запустите установщик и следуйте инструкциям

5. После установки перезапустите PowerShell/Terminal

## Вариант 2: Установить .NET SDK (для разработки)

1. Откройте браузер и перейдите:
   https://dotnet.microsoft.com/download/dotnet/8.0

2. Найдите раздел **".NET SDK 8.0.x"**

3. Скачайте версию для Windows x64

4. Запустите установщик

## Вариант 3: Запустить уже собранный exe

Если папка `bin/Release/net8.0-windows/` уже скопирована на Windows:

1. Перейдите в папку `bin\Release\net8.0-windows\`
2. Запустите `RadioPlayer.exe` напрямую (двойной клик)

**Примечание:** Для работы .exe файла всё равно нужен .NET Desktop Runtime (Вариант 1).

## Проверка установки

После установки откройте PowerShell и выполните:
```powershell
dotnet --version
```

Должна появиться версия, например: `8.0.415`
